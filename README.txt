STOCK MARKET RESEARCH PRO LIVE - GITHUB READY

Isi folder ini sudah lengkap dan siap di-upload ke GitHub.

STRUKTUR:
- index.html
- vercel.json
- .env.example
- api/research.js
- README.txt

CARA UPLOAD KE GITHUB:
1. Extract ZIP ini dulu.
2. Di GitHub klik Upload files.
3. Upload SEMUA isi folder ini sekaligus, termasuk folder api.
4. Commit changes.

CARA DEPLOY KE VERCEL:
1. Login ke Vercel dengan akun GitHub.
2. Import repository ini.
3. Di Environment Variables isi:
   - ADOBE_STOCK_API_KEY = API key Adobe Stock kamu
   - ADOBE_STOCK_PRODUCT = stock-ai-tool
4. Deploy.

CATATAN:
- Jangan upload file ZIP langsung ke GitHub.
- Yang di-upload adalah isi folder setelah di-extract.
- Folder api wajib ikut ter-upload, kalau tidak riset live tidak jalan.
